## Installation

    npm install @textlint-ja/textlint-rule-no-dropping-i

## Usage

Put `@textlint-ja/textlint-rule-no-dropping-i` to `.textlintrc`

```js
{
  "rules": {
    "@textlint-ja/textlint-rule-no-dropping-i": true
  }
}
```
